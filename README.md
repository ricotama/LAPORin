# **LAPORin**

**LAPORin** adalah aplikasi pelaporan kerusakan infrastruktur berbasis **React Native**, 
dengan fitur **foto**, **GPS**, dan **Firebase Realtime Database**.

## ğŸ§© **Komponen Pembangun Produk**

### **1. React Native + Expo**
         Digunakan sebagai framework utama untuk membangun aplikasi mobile yang dapat dijalankan di Android maupun iOS.

### **2. TypeScript**
         Digunakan untuk struktur kode yang lebih aman, kuat, dan terprediksi.

### **3. Firebase Realtime Database**
         Digunakan untuk menyimpan data laporan secara realtime, termasuk:
         * Judul laporan
         * Deskripsi
         * Kategori
         * Lokasi (latitude & longitude)
         * Foto dalam format Base64
         * Timestamp

### **4. React Native Maps**
        Digunakan untuk menampilkan laporan dalam bentuk titik lokasi pada peta.

### **5. Expo Image Picker**
         Digunakan untuk mengambil gambar dari kamera atau gallery.

### **6. Expo Location**
         Digunakan untuk mengambil lokasi GPS perangkat secara otomatis.

### **7. Lucide Icons & Ionicons**
         Digunakan sebagai ikon pada elemen UI aplikasi agar tampilan lebih informatif dan intuitif.
